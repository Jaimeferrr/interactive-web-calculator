# 🧮 Modern Web Calculator

A sleek, interactive calculator built with **Vanilla JavaScript**, CSS3, and HTML5. This was my first project as a developer, focusing on DOM manipulation and event handling logic.

## 🚀 Features
- **Real-time Calculation:** Performs basic arithmetic operations instantly.
- **Smart Correction:** Includes "Clear All" (C) and "Delete" (Backspread) functions.
- **Error Handling:** Gracefully manages invalid mathematical expressions using `try-catch` blocks.

## 🛠 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling and layout.
- **JavaScript (ES6):** Logic, event listeners, and dynamic DOM updates.
